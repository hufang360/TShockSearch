# TShockSearch

#### 介绍
查一查。查物品名，查物品id，查合成。

#### 指令
```
/search <关键词/物品名/id>, 查找物品名/id
/search <id+>, 查找指定id起始的物品，例如 100+ 表示查看101~120的物品
/search list, 列出配置的关键词
/recipe <物品名/id>, 查询物品的合成信息

// 给管理员用
/search add <关键词> <id>, 添加关键词
/search del <关键词>, 删除关键词


// 简写
/search = /s
/recipe = /r
```

#### 权限
```
hf.search
hf.search.admin
hf.recipe
```
